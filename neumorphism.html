Softshell は、あらゆるコントロールをひとつの中立的なマテリアルから押し出します。デュアルシャドウが鋭いエッジを持たずに奥行きを刻み、インターフェースを実際に押せるような感触に仕上げます。

装飾なき、深み。

Softshell のインターフェースを触感豊かに仕上げる3つの原則。光沢なし、グラデーションなし、ギミックなし。

押し出されたコントロール

ボタン、カード、入力フィールドのすべてが同じ面から押し出されます。デュアルシャドウが、ボーダーをひとつも使わずに本物の浮き上がりを生み出します。

触覚的なプレス状態

操作するとシャドウがインセットに反転し、要素が面に沈み込みます。純粋な CSS だけで、ハプティクスのような満足感を実現します。

単一パレットシステム

UI 全体をひとつの背景色で統一。光と影のトーンはアルゴリズムで導出され、コンパクトなウィジェットからフルページレイアウトまでシームレスにスケールします。
