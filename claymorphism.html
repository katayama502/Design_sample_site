デザインを。

やわらかく成形された粘土から生まれた UI システム。ぷっくりと膨らんだ、手で触りたくなるコンポーネントが、インタラクションへと誘います。

角ばった辺もなく、硬いグリッドもない。あるのは純粋な楽しさだけ。

Clay コンポーネントキット
48種の成形可能なコンポーネント。角なし、個性は無限大。
Squish UI の理由

触れるために、作られた。

すべてのコンポーネントは、ぷっくりと膨らんだ粘土を模した多層シャドウで彫刻されています。これは単なる UI ではなく、触覚的なデザインです。

はじめからぷっくり

すべてのシェイプは3層のbox-shadowで膨らみ、奥行きと弾力感をそのまま表現します。設定不要で即使えます。

弾むインタラクション

ホバー時のスケール、クリック時の沈み込み、スプリングトランジションで、すべての操作が温かい粘土を押すような感触になります。

パステルパレット

新鮮な粘土の色を思わせる、やわらかく彩度を落とした厳選カラー。親しみやすく、温かみがあり、自由に組み合わせられます。

実績で語る

世界は粘土を愛している。

Squish UI はすでに、人々が本当に使いたいと思うプロダクトを形作っています。
